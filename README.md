# Project Name: Casey Wormington Banking App

## Project Description
This is the main project for the Advanced C# course. It includes concepts such as Inheritance, Abstraction, Polymorphism, Iteration, Access Specifiers, and Database storage and access. It also helps introduce version control using GitHub.

## Project Tasks
- Plan the project
    - Create an application that uses many different coding concepts throughout the course
- Design the application
    - Take feedback from the project proposal and start designing systems
- Implement the design
    - Take feedback from the initial design and implement the main code without the use of a database yet.
- Database implementation
    - Convert parts of the code to integrate with a local database.
    - The database is created when the application is ran
- Submit application

## Skills Learned
- Database integration
- C#
- Version control
